9. POST-ENCODING CORRUPTION - applyPostEncodingCorruption (Lines 1437-1530)
================================================================================

Preserves first 8 bytes (magic/signature) to ensure decoder attempts parsing

PNG-Specific Mutations (6 strategies):
  0: Corrupt IHDR dimensions (width/height at bytes 16-23)
  1: Corrupt bit depth (byte 24) or color type (byte 25)
  2: Corrupt chunk CRCs (walk chunk structure)
  3: Truncate IDAT stream (zero out trailing data)
  4: Corrupt PLTE entries if present
  5: Random 2% bit flips preserving magic bytes

JPEG/TIFF: Random bit flips only (2% of bytes)
